# Propensity-Score-Methods

- 概要

  - バイアスのある人工データを生成し，下記の手法それぞれで因果効果を推定．結果を比較する．
    1. 単純な群間比較
    2. 傾向スコアマッチング（非復元抽出/復元抽出）
    3. IPW (Inverse Probability Weighting) 法
    4. OW (Overlap Weighting) 法
